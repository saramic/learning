# Tech Talk Review Prompt

Analyse this tech talk video. Return markdown with exactly four sections.

## 1. Three standout segments

Pick one of each flavour where possible. For each: timestamp, label, one sentence on why it landed.

- ⚡ **Energy** — audience most awake, speaker most alive
- 😬 **Despair** — something went wrong, awkward pause, speaker fought through it
- 🏆 **Best moment** — the single clip to post on X; punchy, self-contained, no context needed

## 2. Summary

One paragraph, max 80 words, written for someone who missed the talk. Assume a technical audience.

## 3. Three key takeaways

One sentence each. What should the audience remember tomorrow?

## 4. X post

Ready to paste. Max 280 chars. Include the best moment timestamp. Punchy. Max 2 hashtags.

---

Be direct. If something didn't land, say so. Timestamps as `[mm:ss]`.
