#### Voices and Roles

**The standard toolkit** — every baroque arrangement uses some of these

- **Cantus firmus** — the main melody, fixed and unchanged. Everything else
  decorates around it.
- **Descant** — a florid line soaring above. In hymns it enters for the final
  verse. Synonymous with countermelody in this context.
- **Countermelody** — a composed second melody with its own identity, woven
  against the main tune. More rhythmically independent than a harmony line.
- **Basso continuo** — baroque bass + chords, typically cello/harpsichord.
  Provides harmonic rhythm and root movement. For recorders: your lowest
  instrument carries this.
- **Fauxbourdon** — parallel sixths and thirds below the melody. The simplest
  Renaissance harmony — sounds immediately "old".
- **Imitation / canon** — one voice copies another a few beats or bars later
  (rounds are canons). Very baroque.
- **Pedal tone** — one voice holds a long note while others move. Creates
  tension when the harmony shifts away from it.
- **Sequence** — a melodic figure repeated at successively higher or lower
  pitches. Bach's most overused tool, in the best way.
- **Ornaments** — trills, mordents, turns. Written small but define the baroque
  style. Recorders do these well.

**Typical 4-part recorder layout**

- **Soprano / Descant 1** — melody (cantus firmus)
- **Soprano / Descant 2** — harmony, thirds or sixths below
- **Alto / Treble** — inner voice, fills chord tones, may carry countermelody
- **Tenor or Bass recorder** — bass line, basso continuo role

The "Prokofiev trick": keep this structure intact but subvert the notes inside
it — unexpected notes in expected rhythmic positions.

#### Countermelody

Rules for writing a good countermelody

- **Contrary motion is your friend** — when the melody rises, the counter often
  falls, and vice versa. This creates independence.
- **Fill the rests** — when the melody holds a long note or rests, the
  countermelody moves. When the melody moves busily, the counter can hold. They
  take turns.
- **Start and end on chord tones** — the first and last notes of each phrase
  should be the root, third, or fifth of the current chord. Passing notes in
  the middle are fine.
- **Avoid parallel fifths and octaves** — baroque rule. Two voices moving in
  parallel fifths sounds hollow and was forbidden. Check the interval between
  your counter and the melody at each strong beat.
- **Aim for 6ths and 3rds at strong beats** — these are the sweetest harmonic
  intervals. Avoid unisons (except at the end) and seconds/sevenths on beat 1.
- **Give it a peak** — the countermelody should have its own high point,
  ideally offset from the melody's climax.
- **Same or different rhythm?** — a counter that moves in quavers while the
  melody moves in crotchets creates energy. A counter that moves in longer
  notes sounds serene and "floating". Both are valid.

The "Prokofiev subversion" for countermelodies
- Write a normal countermelody first, then replace one or two notes with the
  chromatic note a semitone away — especially on weak beats.
- Approach a chord tone from the note a semitone above rather than from within
  the scale. Sounds slightly wrong but resolves correctly.
- Use the raised 7th (leading tone) going up, but the flattened 7th going down
  — this is "double inflection" and sounds immediately 20th century while
  staying tonal.

#### Bassline

Basso continuo — the baroque bass baroque

- **Move on chord changes, not every beat** — the bass defines when the harmony
  changes. In 3/4, it often plays on beat 1 only and rests or repeats on 2 and
  3.
- **Root position is stable, first inversion gives forward motion** — a bass
  line that goes root → third of next chord → root creates smooth voice
  leading.
- **Walking bass** — the baroque workhorse. Fill the gap between two chord
  roots with scale steps or passing tones. E.g. C → G: bass walks C B A G
  (stepwise descent).
- **Ground bass / ostinato** — a repeating bass pattern (like Pachelbel's
  Canon) over which the upper voices vary freely. Very effective for recorder
  arrangements.
- **Avoid leaping to a dissonance** — the bass can leap, but land on a chord
  tone, not a 7th or 9th.

Modern bass approaches modern

- **Pedal point** — hold the tonic or dominant in the bass while the upper
  voices change chords above it. Creates ambiguity and tension.
- **Bass line in contrary motion to melody** — when the melody climbs, drop the
  bass. Prokofiev does this constantly.
- **Chromatic bass descent** — one of the most effective tricks: bass descends
  chromatically (e.g. G → F# → F → E → Eb) while upper voices sustain or move
  slowly. Creates inevitable forward pull.
- **Avoid doubling the root when the chord is already stable** — the bass can
  play the fifth instead, giving a lighter, more ambiguous feel (used heavily
  by Ravel).

#### Recorder harmony

Traditional recorder harmony — the rules baroque

- **Stay in range** — descant: D5–D7 practical (C5–E7 possible). Alto/treble:
  F4–G6. Tenor: C4–D6. Bass recorder: F3–G5.
- **Thirds and sixths** — the default interval between voices. Parallel thirds
  under the melody sound immediately "folk" and satisfying.
- **Fauxbourdon** — three voices: melody on top, a voice a third below, another
  a sixth below (= fourth below the middle). Pure Renaissance texture.
- **Voice crossing is allowed but use carefully** — a lower voice can
  momentarily go above a higher one for melodic reasons, but it muddies the
  texture if overdone.
- **Double the root in four-part writing** — if in doubt, two voices can play
  the root, one the third, one the fifth.
- **Avoid doubling the leading tone** — the 7th degree of the scale (the note a
  semitone below the tonic) should not be doubled; it wants to resolve upward
  and two voices resolving to the same note sounds clunky.
- **Prepare and resolve dissonances** — a 7th or 4th in a chord should be
  approached by step and leave by step (usually downward). This is the whole of
  baroque voice-leading in one sentence.

Practical shorthand for 4-part recorder texture

- Soprano 1: melody note
- Soprano 2: diatonic third below (or sixth below for variety)
- Alto: fifth of the chord, or the note that completes the triad
- Bass/Tenor: root of the chord

Then apply the Prokofiev rule: occasionally swap one of those diatonic
intervals for its chromatic neighbour.

#### Chrods

Traditional chord progressions — what sounds baroque

- **I → IV → V → I** — the foundation of everything. All other progressions are
  elaborations.
- **Cadences:** perfect (V→I), plagal (IV→I "amen"), half (anything→V),
  deceptive (V→vi — ends on the relative minor instead of tonic, a surprise).
- **Circle of fifths descent** — I vi ii V I (or with 7ths: Imaj7 VIm7 IIm7 V7
  I). Bach's default. Sounds inevitable and satisfying.
- **Sequence** — the same chord pattern repeated a step lower or higher: I V vi
  iii IV I IV V. Used constantly in baroque.
- **Neapolitan chord (bII)** — a major chord built on the flattened second
  degree. In C minor: Db major. Very dramatic, signals a strong cadence coming.
- **Diminished seventh** — the most tense chord in the tonal system. Four notes
  each a minor third apart. Works anywhere as a pivot chord.

Modern chord transitions modern

- **Chromatic mediant** — chords whose roots are a third apart and share one
  common tone, e.g. C major → Eb major → Ab major. Sounds film-score, very
  Prokofiev. No voice-leading rules — just slide.
- **Parallel planing** — move a chord shape up or down chromatically while
  keeping its quality (e.g. all major chords). Debussy / Ravel technique.
  Breaks every voice-leading rule and sounds gorgeous.
- **Modal mixture** — borrow a chord from the parallel minor (or major). In C
  major: use Fm (iv from C minor), or Ab major (bVI), or Bb major (bVII). Gives
  instant colour without leaving the key.
- **Secondary dominant** — a V7 chord applied to any scale degree, e.g. A7 in C
  major (V of ii = V/ii). Adds brief tonicisation without a full modulation.
- **Tritone substitution** — replace V7 with the chord whose root is a tritone
  away (e.g. replace G7 with Db7 in C major). Both share the same tritone
  interval (B/F) but approached from the opposite side. Jazz technique that
  sounds very 20th century even in a baroque texture.

#### Modes

- Ionian = major
  - Bright, stable, resolved. Your default starting point.
  - `W W H W W W H`
- Dorian minor + ♮6
  - Minor but with a raised 6th — the "Celtic folk" mode. Scarborough Fair,
    Drunken Sailor. Great for recorder music — already feels historical.
  - `W H W W W H W`
- Phrygian minor + ♭2
  - Dark, Spanish, dramatic. The ♭2 is its fingerprint — sounds ancient and
    modal. Use for a verse that needs edge.
  - `H W W W H W W`
- Lydian major + ♯4
  - Major but dreamy, floating, slightly unresolved. The ♯4 is the magic note —
    sounds like something is "about to happen". Film composers love it.
  - `W W W H W W H`
- Mixolydian major + ♭7
  - Major but with a flattened 7th — sounds "folk/rock" or "bluesy". Sweet Home
    Alabama. The V chord becomes minor, which is the tell.
  - `W W H W W H W`
- Aeolian = natural minor
  - Your default minor. Sad, introspective. The harmonic minor adds the leading
    tone (♯7) for stronger cadences.
  - `W H W W H W W`
- Locrian ♭2 + ♭5
  - Unstable, deeply dissonant — the tonic chord itself is diminished. Almost
    never used as a home key, but brilliant for a momentary destabilisation.
  - `H W W H W W W`

How to use modes for your "baroque but not" effect

- **Start in G Dorian (G A Bb C D E F G)** — indistinguishable from G minor to
  a casual ear, but the natural E gives it a brighter folk character. Your
  Laudate Dominum is already in this zone.
- **Slip into G Mixolydian for one phrase** — change the F# to F natural.
  Suddenly the V chord (D) becomes Dm and the cadence feels unresolved. Very
  effective for "something's slightly off" feeling.
- **Touch Lydian for a moment of wonder** — raise the 4th degree by a semitone
  for one bar. In G: C → C#. The melody needs to land on it; don't bury it in
      harmony.
- **The Prokofiev trick with modes** — write in major but briefly switch the
  3rd or 7th in one voice only, while the other voices stay major. Creates a
  subtle clash that resolves immediately.

#### Key Changes

The Jacob Collier harmonic descent — what's happening

The Instagram reel you linked shows a classic "chromatic mediant + voice
leading" trick. The crowd sings a chord, it shifts down by semitone, then
resolves. Here's the anatomy:

```
C A G♭ E♭ = C major 7 (with E♭ = ♭3 = chromatic colour note)

  ↓ all voices drop a semitone (chromatic slide)

B A F♯ D♯ = B major 7

  ↓ resolve upward (voice leading pulls to G# minor)

B G♯ E = E major (or G♯ minor) — the "home"
```

The trick: the first two chords are chromatic mediants — not diatonic
neighbours. Moving all voices in parallel by a semitone breaks every classical
voice-leading rule but sounds perfectly smooth because the chord quality is
preserved. The final resolution to E major works because B is the dominant of
E.

Key changes you can use in recorder music techniques

- **Pivot chord modulation** — find a chord that exists in both keys, use it as
  the bridge. E.g. Am is both vi in C major and i in A minor. Most natural,
  least jarring. Baroque standard.
- **Chromatic (direct) modulation** — just jump. End a phrase, start the next
  phrase in a new key with no preparation. Works when the phrase ending is
  strong. Very Prokofiev — he does this constantly in the Classical Symphony.
- **Enharmonic modulation** — use the fact that some chords sound the same but
  are "spelled" differently. Diminished 7th chords are the easiest: the same
  four notes can belong to four different keys, so you can walk in one key and
  walk out of a completely different one.
- **Chromatic mediant shift (the Collier move, recorder-friendly version)** —
  go from your home chord to a chord whose root is a major or minor third away
  and shares one common tone. E.g. C major → Eb major (both have G). Then
  resolve from Eb back to C via a dominant. One held note in one voice + two
  moving voices creates the effect without needing 4 independent parts.
- **Semitone shift of the entire texture** — the Collier parallel slide. All
  four recorder parts move together by a semitone. Sounds cinematic. Works best
  for one bar, then you resolve somewhere logical.
- **Mode mixture modulation** — stay on the same tonic but shift mode. G minor
  → G major → G Dorian. No new root, but the colour shifts completely. Very
  achievable on recorders.

#### Baroque and Modern

The Classical Symphony principle — "sounds baroque, isn't"

Prokofiev's Symphony No. 1 (1917) follows every baroque formal rule — two-bar
phrases, sequences, harpsichord-like texture, ornaments — but injects three
specific distortions. You can apply all three to recorder music.

Distortion 1: unexpected chromatic notes trick

- Take a perfectly diatonic melody and replace one note with its chromatic
  neighbour — usually the note a semitone above or below a chord tone.
- The surrounding notes are correct; the "wrong" note is surrounded by right
  ones, so it sounds surprising but not atonal.
- Best on a weak beat or as a passing note, not on beat 1 of a strong bar.
- Example: in G major, a melody going G A B C — replace the C with B# (= C
  natural, or in recorder terms, add a C# instead). The ear expects C, gets C#,
  and the phrase resolves back to D normally.

Distortion 2: wrong-key harmonisation trick

- Harmonise a melody note with a chord from a completely different key — then
  immediately return to the home key.
- Prokofiev is famous for putting an Eb chord under a G major melody. The E♭
  lasts one bar, the next bar is back to G major. The melody note (G) is common
  to both chords — that's what makes it work.
- For recorders: the top voice holds the melody note, the lower three voices
  slip to the "wrong" chord, then all return. A single common tone creates the
  bridge.

Distortion 3: abrupt register jumps trick

- Take a melodic phrase and continue it an octave higher or lower than expected
  — the "right" notes, "wrong" register.
- Haydn used this for comedy; Prokofiev uses it for the feeling of something
  slightly surreal.
- For recorders: your soprano recorder going from its middle register to its
  piercing upper octave mid-phrase is the most effective version of this.

The three modes most useful for your recorder aesthetic

- **Dorian** — already sounds authentically early music. Renaissance and
  baroque composers used it constantly. Your Laudate Dominum is Dorian. Lean
  in.
- **Mixolydian** — the flattened 7th sounds archaic (modes predate the
  major/minor system). One bar of Mixolydian in a major piece sounds like a
  temporary step into old church music.
- **Lydian for a moment of "lift"** — raise the 4th in one voice for one bar
  when you want a sudden brightening. Sounds like a shaft of light. Then return
  to normal major.

The recipe: baroque recorder piece with modern edge

- Write the melody and harmonies as if you are Telemann — diatonic,
  sequence-heavy, ornamental.
- Then: in every 8-bar phrase, pick one note in one inner voice and make it
  chromatic. Just one.
- In the B section, use a chromatic mediant chord change instead of the
  expected V→I.
- In the final phrase, do one Collier-style semitone slide of all four voices
  for one beat, then resolve to tonic.
- End with a Picardy third if in minor — the unexpected major chord at the end
  of a minor piece. This is authentically baroque AND slightly startling.

#### Switch to an MCP workflow

The best one right now is ghchen99/mcp-musescore on GitHub — it provides
programmatic control over MuseScore via a WebSocket-based plugin system,
allowing Claude to compose music, add lyrics, navigate scores, and control
MuseScore directly. Wikipedia

How it works: it functions as an MCP server integrating with MuseScore through
a WebSocket plugin, bridging AI models and the notation software. You install a
.qml plugin into MuseScore, run the Python server locally, and then connect
Claude Desktop to it via your claude_desktop_config.json. PyPI
What it can do:

Add notes and rests, select and delete notes, create tuplets, undo changes, and
switch between voices. Wikipedia

Navigate to specific measures, move cursor element by element, and select
custom ranges across staves. Wikipedia

There's also a Korean-built one — strongbeen04/MUSESCORE-MCP wraps the
MuseScore CLI, enabling format conversion (PDF, PNG, MusicXML, MIDI, MP3), part
extraction, and harmonic analysis including Roman numeral analysis. Song of
America

The catch: these run on Claude Desktop (the local app), not claude.ai in the
browser, because they need to connect to a locally running MuseScore instance
on your Mac. The setup is:

Install Claude Desktop (free, works with your account)
- Clone the GitHub repo, install Python dependencies
- Drop the .qml plugin file into your MuseScore plugins folder
  (~/Documents/MuseScore4/Plugins/)
- Add the server config to ~/Library/Application
  Support/Claude/claude_desktop_config.json
- Open MuseScore with a score, enable the plugin, and then Claude Desktop can
  directly edit it

This would completely change the workflow we've been doing — instead of me
generating MusicXML for you to import, I could write harmonies directly into an
open MuseScore score in real time. Worth setting up if you're going to keep
doing this kind of work.
